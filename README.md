

## Description

A simple application that uses NestJS & Serverless framework to create CRUD operation of an Employee entity.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm  start


```

## Deploy

```bash
$ serverless deploy --stage dev


```
